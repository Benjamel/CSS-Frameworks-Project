## Description

CA for CSS Framework. Made with bootstrap only

## Built With

- HTML/CSS
- Bootstrap
- Scss

### Installing

1. Clone the repo:

```bash
git clone https://github.com/Benjamel/CSS-Frameworks-Project.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run watch
```
